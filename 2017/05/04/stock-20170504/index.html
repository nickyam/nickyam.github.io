股海泛舟，扬帆起航

发表于 2017-05-04 | 更新于 2017-05-07 | 分类于 投资

常言道：五穷六绝七翻身，八下九回头十空。对股民来说，5月是充满挑战的月份。作为一个立志要在股市里安身立命的小散，从今天开始每日记下股市心得，积跬步以至千里。过往的荣辱抛诸脑后，轻装上阵，股海泛舟，扬帆起航。

浙大网新

这支股票是我精心挑选出来准备长期关注的股票。

一、介入原因

  1. 血统：浙江的股票，支持家乡；浙大的关系，有科研基础；
  2. 指标：市值小于150亿元，市盈率50以下
  3. 股东：社保基金从去年三季度大举进入
  4. 前景：阿里的合同，榜上大树

二、具体操作

既然选定了一支股票，并且技术面尚可，那就轻仓介入。

  • 3-27 介入（14.58）随后几日跌破20日均线
  • 3-30 加仓（13.56）随即反弹
  • 再次调整时加仓
  • 跌破均线（后文均线均指20日均线，除特别说明外）继续加仓
  • 未及时止损，浮亏30%
  • 4-27 触及11.29后反弹，轻仓做套
  • 4-28 年报显示社保基金未退出
  • 4-29 季报显示社保基金有加仓
  • 技术面未明显改善，未加仓
  • 5-04 停牌（重大事项），并爆出和阿里蚂蚁金服的合作，利好。（后悔未加仓）

三、后市操作

  • 时刻关注复牌动向
    • 首日集合竞价高开出，当日低点接回
    • 首日集合竞价平开或低开，加轻仓
  • 严格技术面操作（压力和支撑），特别是止损操作

伟星新材

这是我在白马股大涨中观察的一支股票，但是一直没有介入。

业绩公布不及预期，4月27日大跌，跌停板介入。

一、介入原因

  1. 短线操作：跌停板恐慌溢价
  2. 均线理论：一日回调至均线

（未关注到macd死叉）

二、具体操作

  • 次日低开，集合竞价未出
  • 盘中恐慌至跌停，未做套
  • 总结一句话，短线操作

五一节去新房考察，发现水管用的就是伟星的产品，感觉质量还行。节后伟星没有加仓，这几天错过行情。

三、后续操作

  • 严格按照技术面操作

广州港

最近白马股回调严重，一带一路熄火，雄安因为大资金被套我也没有介入，所以我把目光投向之前做的次新股。

  • 昨天次新股集体爆发，我轻仓介入广州港（10.38），当日涨停（11.21）。
  • 今早低开，优柔寡断，集合竞价（10.95）未出，最终被洗盘（10.70）。

为什么会被洗？因为上一日的封板是因为主力的高换手，今天成交量没有，料后市无表现。但是下午被打脸（翻红最高至11.62）。与之前打板坤彩科技如出一辙。心理素质不行，以后少做这一类。
