什么是静态网站？完全静态的网站只有html文件，对服务器资源消耗极低，可以免费托管在 Github Page 等。对于学生等个人站长而言，运行成本非常低。此外，静态网站安全性很好，几乎没有什么可攻击的漏洞，即使恶意访问量很大，静态网站也可以很好地支撑。所以不少站长选择从 Wordpress 转移到 静态网站。
